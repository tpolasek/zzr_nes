# ZNES - NES Emulator written in rust.

# Key planned features include:
- Cycle accurate PPU and CPU emulation
- Minimalistic design and boilerplate code

# Completed:
- 6502 CPU emulation
- Graphic Framebuffer
- 1 Controller input
- Rom Loading
- Basic PPU core
- Basic Debugger
